# Passport Application Experience Redesign

## Project Overview
This is a simple frontend redesign of the passport application process.  
The goal of this project is to make the process easier and clearer for first-time users.

## Features
- Home page
- Login page
- Dashboard
- Application form
- Document upload page
- Appointment booking page
- Confirmation page
- Draft save using localStorage

## Why I Made This
Many users find passport application confusing because of long forms and unclear steps.  
In this project, I tried to make the flow simple, step-by-step, and easy to understand.

## Tech Stack
